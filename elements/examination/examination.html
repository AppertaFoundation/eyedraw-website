Description:

The clinical examination event allows entry of a large range of information about a clinical examination. The information is broken down into a series of elements, the initial display of which is dependent on the specialty (and sub-specialty) of the user, the site of installation, and the status of the patient. This allows the event to be customised to show only the required elements when appropriate.

However, there will be occasions when a user wishes to add information unexpectedly. In this circumstance, the available elements are made available by clicking on the link at the left of the page.

In this example, there are three elements, the first and third of which are displayed by default. For the purposes of this demonstration, only one data field is shown for each element. In a working version, the relevant element (see elsewhere on the website) would be fully displayed.

Only information visible in open elements is saved, and then displayed when in display mode.
