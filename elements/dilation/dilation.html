Description:

This element handles the recording of dilating drops for use in an ophthalmic examination. This element will mainly be used as part of the ophthalmic examination event. It addresses items 3.61 to 3.64 of the Glaucoma national dataset.

Selection of the dropdown creates a time stamp, from which (if the date is the same day), the length of time since dilation can be calculated and displayed. This should be useful to medical staff when deciding which patients to see next in a clinic.

The contents of the drop down menu are determined at the sub-specialty level. Selecting 'Other' prompts for the name of the drug.
