Description:

This element collects outpatient procedures, and will be an optional element in the examination event. This version simply collects the fact that the procedure was done, without any significant 'metadata'. It will replace the current paper form, and have the dual function of recording a clinical procedure, and registering it for payment (the sole purpose of the current paper form).

The current content is a mixture of the procedures mentioned in recent emails, and the current paper form, and can be easily changed. The procedures offered will also be sub-specialty specific.

This element would be shared with all other sub-specialties that carry out procedures in outpatients, so needs to be as generic as possible. There is one text box to provide for any additional clinical information.

NB laser procedures are collected using the current laser event modified to include sub-specialty defined laser procedures.
